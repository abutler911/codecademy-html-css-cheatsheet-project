Common CSS Selectors

Selector Type Selector Syntax What It Does Specificity
ID-Class-Tag-Pseudo
Universal Selector * Selects all elements. 0-0-0-0
Class .classname Selects all elements with the given class name. 0-1-0-0
ID #idname Selects the element with the given ID. 1-0-0-0
Attribute Selector [attribute] Selects all elements with the specified attribute. 0-1-0-0
Attribute Value Selector [attribute="value"] Selects all elements with the specified attribute and value. 0-1-0-0
Child Selector parent > child Selects all direct child elements of the parent. 0-0-2-0
Descendant Selector ancestor descendant Selects all descendants of the ancestor element. 0-0-2-0
Adjacent Sibling Selector element1 + element2 Selects all siblings that share the same parent. 0-0-2-0
Pseudo-class Selector ::pseudo-element Styles parts of elements, like ::before or ::after. 0-0-0-1
Negation Pseudo-class :not(selector) Selects all elements that do not match the given selector. 0-1-0-0
First Child Pseudo-class :first-child Selects the first child element of its parent. 0-1-0-0
Last Child Pseudo-class :last-child Selects the last child element of its parent. 0-1-0-0
Nth-child Pseudo-class :nth-child(n) Selects the nth child element of its parent. 0-1-0-0
