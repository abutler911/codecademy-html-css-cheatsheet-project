Common CSS Selectors

Selector Type Selector Syntax What It Does
Universal Selector * Selects all elements.
Class .classname Selects all elements with the given class name.
ID #idname Selects the element with the given id.
Attribute Selector [attribute] Selects all elements with the specified attribute.
Attribute Value Selector [attribute="value"] Selects all elements with the specified attribute and value.
Child Selector parent > child Selects all direct child elements of the parent.
Descendant Selector ancestor descendant Selects all descendants of the ancestor element.
Adjacent Sibling Selector element1 + element2 Selects all siblings that share the same parent.
Pseudo-class Selector ::pseudo-element Styles parts of elements, like ::before or ::after.
Negation Pseudo-class :not(selector) Selects all elements that do not match the given selector.
First Child Pseudo-class :first-child Selects the first child element of its parent.
Last Child Pseudo-class :last-child Selects the last child element of its parent.
Nth-child Pseudo-class :nth-child(n) Selects the nth child element of its parent.
